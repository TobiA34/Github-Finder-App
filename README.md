
# Github Finder 
 
 This project allows you to search for GitHub accounts and see the person's repo.



## Deployment

To deploy this project run

```bash
  npm start 
```




## Features

- Light/dark mode toggle
- Live previews
- REST API
- Validation
- Responsive
